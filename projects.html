Linker - C

Description: Personal Project

  • Developed a linker in C to combine multiple LC2K object files into a single executable.
  • Implemented concatenation of text and data segments, preserving the order of input files.
  • Handled relocation entries to fix references and produce a unified machine code file.
  • Followed the LC2K convention of executing code starting from the first file provided as an argument.
  • Incorporated handling of the Stack label and addressed peculiarities in LC2K programming.
  • Ensured error checking for duplicate defined global labels, undefined global labels, and the presence of the Stack label in object files.
